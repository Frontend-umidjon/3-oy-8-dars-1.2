The Fundamentals of If-Else Statements

1

If Statement

The simplest form. It executes code if a condition is true.

2

If-Else

Adds an alternative. It handles both true and false scenarios.

3

Else-If

For multiple conditions. It allows you to check several possibilities.

Leveraging Boolean Logic

AND (&&)

Both conditions must be true. It's like a strict gatekeeper.

OR (||)

Either condition can be true. It's more flexible.

NOT (!)

Flips the condition. It's the rebel of boolean logic.

Code Examples

If Statement

Checks a single condition. Executes code if the condition is true.

" if (age >= 18) {
console.log("You are eligible to vote."); } "

Else Statement

Provides an alternative path. Runs code if the initial condition is false.

" if (age >= 18) { console.log("You are eligible to vote."); } else { console.log("You are not eligible to vote yet."); } "

Else If Statement

Allows for multiple checks. Evaluates conditions sequentially.

" if (score >= 90) { console.log("A"); } else if (score >= 80) { console.log("B"); } else if (score >= 70) { console.log("C"); } else { console.log("D"); } "
